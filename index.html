Deploy & share Flightwatch

Your routes and history are saved in this browser. To access from any device or share the app, host it free on GitHub Pages:

  1. Go to github.com → New repository → name it flightwatch
  2. Click Add file → Upload files and upload index.html
  3. Go to Settings → Pages → Source: Deploy from branch → main
  4. Your live URL: yourusername.github.io/flightwatch

Bookmark the URL. It's yours forever, free. Export your saved data below to move it between devices.
